# LibraryProject

## Django Project Setup

This project was created using:

python -m django startproject LibraryProject

## Steps followed

1. Installed Django using pip
2. Created project using python -m django
3. Started development server
4. Verified default Django welcome page

## Run Server

python manage.py runserver

Then open:
http://127.0.0.1:8000/

## Project Structure

manage.py → Django command utility  
settings.py → Project configuration  
urls.py → URL routing  
wsgi.py → Deployment entry
